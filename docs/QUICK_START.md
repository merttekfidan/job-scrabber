## Troubleshooting

### Extension doesn't appear
- Make sure Developer mode is enabled
- Try refreshing the extensions page
- Check that you selected the correct folder

### "Please set your Gemini API key"
- Open Settings in the extension popup
- Paste the API key exactly as shown above
- Click Save

### "Failed to extract data from page"
- Make sure you're on an actual job posting page (not search results)
- Try selecting some text from the job description first
- Some sites may have unusual layouts - try a different job board

### "Failed to process job data"
- Check your internet connection
- Verify the API key is saved correctly
- The free tier has rate limits - wait a minute and try again
